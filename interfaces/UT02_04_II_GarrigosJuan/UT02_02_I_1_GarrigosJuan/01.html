solid: Defines a solid border

double: Defines two borders. The width of the two borders are the same as the border-whidth value

groove: defines 3D grooved border. The effect depends on the border-color value

ridge: Defines a 3D ridge border. The effect depends on the border-color value

inset: Defines a 3D inset border. The effect depends on the border-color value

outset: Defines a 3D outset border. The effect depends on the border-color value
